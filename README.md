- `about.html` الرؤية والأهداف
- `cases.html` حالة الوالد حمد
- `contact.html` التواصل
- `workshops.html` صفحة الورش والتسجيل
- `style.css` التصميم
- `script.js` القائمة والتفاعلات
- `images/` الصور والشعار

## النشر على GitHub Pages
1. أنشئ مستودعًا جديدًا في GitHub.
2. ارفع جميع الملفات كما هي.
3. افتح Settings ثم Pages.
4. اختر Deploy from a branch.
5. اختر فرع `main` والمجلد `/root`.
6. اضغط Save.
