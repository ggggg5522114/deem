# موقع فريق ديم الخيري

موقع عربي متجاوب، جاهز للرفع على GitHub Pages.

## الملفات
- `index.html` الصفحة الرئيسية
- `about.html` الرؤية والأهداف
- `cases.html` حالة الوالد حمد
- `contact.html` التواصل
- `workshops.html` صفحة الورش والتسجيل
- `style.css` التصميم
- `script.js` القائمة والتفاعلات
- `images/` الصور والشعار

## ملاحظة
حدّث أرقام التبرعات في `index.html` و`cases.html` عندما تتغير بيانات الحالة.

## النشر على GitHub Pages
1. أنشئ مستودعًا جديدًا في GitHub.
2. ارفع جميع الملفات كما هي.
3. افتح Settings ثم Pages.
4. اختر Deploy from a branch.
5. اختر فرع `main` والمجلد `/root`.
6. اضغط Save.

## التوافق
- تصميم متجاوب مع الهاتف والتابلت والكمبيوتر.
- قائمة هاتف منسدلة.


## تحسينات الإصدار الجديد
- حركات ظهور خفيفة باستخدام Intersection Observer مع احترام إعداد تقليل الحركة.
- عناوين وأوصاف SEO مخصصة لكل صفحة.
- بيانات Open Graph وTwitter لتحسين شكل الروابط عند المشاركة.
- بيانات منظمة Schema.org من نوع NGO.
- ملف robots.txt للسماح بأرشفة الموقع.
- تحسين أوصاف الصور وتحديد أبعاد الشعار وتقليل تغيّر التخطيط.

## خطوة مهمة بعد اختيار رابط الموقع
أضف رابطًا أساسيًا Canonical لكل صفحة، وأنشئ sitemap.xml بعناوين الموقع النهائية، ثم أضف الموقع إلى Google Search Console.
